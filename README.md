# 蜜语 - 天然蜂蜜水产品独立站

## 项目简介
蜜语是一个销售天然蜂蜜水产品的独立站，主打产品为四种口味的蜂蜜水：柠檬汁蜂蜜水、百香果汁蜂蜜水、黑加仑汁蜂蜜水和青金桔汁蜂蜜水。网站的设计亮点是展示了产品的独特包装设计——蜂蜜和水分开储存，拧开盖子后蜂蜜才会流下去与水混合。

## 技术栈
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- GSAP 动画库
- Font Awesome 图标库

## 功能特点
1. 响应式设计，适配各种设备屏幕
2. 蜂巢和蜜蜂主题的沉浸式首页设计
3. 产品展示区，展示四种不同口味的蜂蜜水
4. 产品动画演示，展示蜂蜜和水分开包装的独特设计
5. 购物车功能，支持添加、删除、调整数量等操作
6. 结算功能，支持填写收货信息和选择支付方式

## 项目结构

## 本地运行
直接在浏览器中打开 `index.html` 文件即可查看网站。

## 部署说明
可以将项目文件上传到任何静态网站托管服务，如GitHub Pages、Netlify、Vercel等。

## 设计亮点
1. 金黄色为主色调，搭配自然绿色和白色，营造天然、健康的视觉感受
2. 蜂巢和蜜蜂主题的背景设计，增强品牌形象
3. 产品卡片悬停放大效果，提升用户体验
4. 蜂蜜流动动画，直观展示产品的独特设计
5. 平滑滚动和页面过渡效果，增强网站的流畅性
